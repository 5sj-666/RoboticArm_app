# RoboticArm_app

机械臂的上位机软件

打开模拟器：open -a simulator。 
运行：   
debugger模式：   
    flutter run    
    输入 r ： reload 热重载  
    输入 R : restart 热重启   

release模式（安装到本地，七天内可用）：    
    flutter build ios   
    flutter run --release   
    首次安装需在手机的设置 -》 通用 -》vpn与设备管理 中授权信任

添加依赖包：
flutter pub add three_js_helpers

其他：
pod install
pod --version
pod install flutter

auto_route: 
    https://pub.dev/packages/auto_route
    脚本生成路由文件：dart run build_runner build
